## 物語構造の選択 (Narrative Structure Strategy)
この物語に最も適した構造を以下から1つ選択（または独自に採用）し、それに基づいてマスタープロットを構築してください。物語構造と章構成は一致する必要はなく、章は最大5章とします（5章を使い切れということではありません。引き算の美学もあります。短編として1章構成にしても構いません。）。

A. **Story Circle (Dan Harmon)**:
   - キャラクターの内的変化と「代償」に焦点を当てる場合。
   - 構成: You -> Need -> Go -> Search -> Find -> Take -> Return -> Change

B. **Kishōtenketsu (起承転結)**:
   - 驚きや、対立構造に依存しない展開を作りたい場合。
   - 構成: Introduction -> Development -> Twist (Recontextualization) -> Conclusion

C. **Fichtean Curve**:
   - スリル、サスペンス、あるいは冒頭から激しい展開にする場合。
   - 構成: Inciting Incident -> Crisis 1 -> Crisis 2 -> ... -> Climax -> Resolution
